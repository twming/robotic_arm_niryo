ROS Stack overview
=========================================

Ned is a robot based on Raspberry, Arduino & ROS. It uses ROS to
make the interface between Hardware and high-level bindings.

On the following figure, you can see a global overview of Niryo's robot software.
It will help you understand where are placed each part of the software.
